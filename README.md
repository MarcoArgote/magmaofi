# Magma Studio

Single-page site for the Magma Studio recording and video space. Built with
React, Vite, and TypeScript.

## Sections

- Inicio (hero with scroll scaling photos)
- Catalogo (cards with checks and pricing)
- Nosotros
- Contactanos (WhatsApp button)
- Login / Register (informative placeholders)

## Animations

- GSAP reveal animations for section content
- Scroll-linked scaling on the two hero photos

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
